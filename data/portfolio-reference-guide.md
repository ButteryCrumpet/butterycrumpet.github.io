# Portfolio Reference Guide - Local Chiba Builders

Visual reference guide for reviewing completed projects and design aesthetics of 11 local and regional house builders in the Kashiwa/Nagareyama/Chiba area.

## How to Use This Guide

This document provides direct links to portfolio galleries and specific project examples for each builder. Use this guide to:

1. **Quickly assess visual aesthetic** - See if their style matches your preferences
2. **Evaluate natural light integration** - Look for courtyards, large windows, open plans
3. **Review natural materials usage** - Assess wood quality, finishes, textures
4. **Understand design range** - See variety from modern to traditional
5. **Find local examples** - Identify projects in your target area

**Recommended Approach:**
- Start with builders matching your top priorities (see Quick Reference below)
- Review 3-5 portfolio examples from each to understand their range
- Note which aesthetics resonate with you
- Schedule consultations with top 3-5 based on visual appeal + technical capabilities
